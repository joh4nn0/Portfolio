Application de gestion d'absence

Cette application a été développée durant ma première année de BUT informatique, c'était le premier vrai projet en C que j'ai réalisé. L'application permet de gérer les absences des étudiants dans un établissement scolaire. Nous avons réalisé ce projet en groupe de 2.

Projet labyrinthe

Dans le cadre de la SAE 2.02, nous avons travaillé sur les graphes et les labyrinthes en Java. Le projet était divisé en deux grandes parties : Trouver les plus courts chemins dans un graphe à l'aide de l'algorithme de Dijkstra. Appliquer cet algorithme à des labyrinthes et afficher le résultat avec une animation graphique.

Technologies utilisées : Java
