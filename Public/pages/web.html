Site Orientation

Ce site a été réalisé à l'aide de tous les étudiants de STI2D de mon année, dans le cadre d'un projet commun. Le but du site est de représenter un maximum de filières d'orientation post-bac pour les lycéens de STI2D. Durant le projet, nous avons dû être divisés en plusieurs groupes ; chaque groupe était chargé d'une technologie (HTML, CSS, PHP, SQL). Chaque étudiant a donc réalisé un site sur un établissement, comme moi avec l'IUT d'Orsay. Puis nous avons mis tous les sites en commun à l'aide du PHP. J'étais, moi, en charge du CSS et du HTML de chaque site ; je devais les mettre sous la même charte graphique. Mes autres camarades devaient, eux, faire soit un questionnaire visant à aider les lycéens dans leur choix d'orientation, soit la partie admin du site. Et d'autres devaient faire des tableaux à l'aide de requêtes SQL pour regrouper différentes informations. Malheureusement, le site n'a jamais été fini par faute de temps.

Technologies utilisées : HTML, CSS, PHP, SQL
