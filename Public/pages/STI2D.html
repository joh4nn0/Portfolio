Site Orientation

Ce site a été réalisé a l'aide de tout les etudiands de STI2D de mon année dans le cadre d'un projet commun. Le but du site est de représenter un maximum de filière d'orientation post bac pour les lycéens de STI2D. Durant le projet nous avons du etre divisé en plusieur groupe chaque groupe etait charger d'une technologie (html, css, php, sql) Chaque etudiant a donc réalisé un site sur un etablissement comme moi avec l'IUT d'orsay. Puis nous avons mis tout les sites en commun a l'aide du php. J'etait moi en charge du css html de chaque site je devait les mettre sous la meme chartre graphique. mes autres camarade devais eux faire sois un questionnaire visant a aider les lycéens dans leur choix d'orientation sois la partie admin du site. et d'autre devait faire des tableau a l'aide de requette sql pour regrouper differente information. Malheuresement le site n'a jamais été fini par faute de temps.

Technologies utilisées : HTML, CSS, PHP, SQL
